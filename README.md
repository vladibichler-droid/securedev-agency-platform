# SecureDev Agency Platform

SecureDev Agency Platform ist ein professionelles Fullstack-Projekt für eine fiktive Developer-Firma.

Das Projekt wird Schritt für Schritt aufgebaut und soll später wie ein echtes Entwickler-Firmenprojekt wirken.

## Ziel des Projekts

Dieses Projekt soll zeigen, dass moderne Webentwicklung aus mehreren Bereichen besteht:

- Frontend
- Backend
- API
- Sicherheit
- Dokumentation
- Projektstruktur
- später Kundenportal
- später Adminbereich

Es ist kein Simulator-Projekt.

Es ist ein echtes Fullstack-Webprojekt mit professioneller Struktur.

## Geplanter Technik-Stack

Frontend:

- React
- TypeScript
- Vite

Backend:

- Node.js
- Express
- TypeScript

Security:

- Sicherheits-Header
- CORS
- Rate-Limiting
- Eingabevalidierung
- Passwort-Hashing
- sichere Fehlermeldungen
